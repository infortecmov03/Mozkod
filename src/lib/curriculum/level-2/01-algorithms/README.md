# ⚡ Algoritmos e Eficiência

Módulo focado em resolução de problemas e análise de complexidade.

## 📁 Estrutura de Laboratórios
Os laboratórios estão divididos entre:
- `javascript`: Focado em lógica de frontend e node.
- `python`: Focado em lógica de dados e backend.

## 🛠️ Como adicionar novos desafios
1. Cria a lição teórica em `theory/`.
2. Cria o exercício na pasta da linguagem desejada dentro de `laboratory/`.
3. Associa a teoria ao exercício no objeto `practice` do `index.ts`.

## 📈 Big O e Performance
Ao criar lições teóricas, utiliza tabelas HTML para comparar a complexidade dos algoritmos, facilitando a visualização do aluno.
