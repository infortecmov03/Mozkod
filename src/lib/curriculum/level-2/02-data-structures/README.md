# 📊 Data Structures (Estruturas de Dados)

Estudo da organização física e lógica dos dados na memória.

## 📁 Organização Multi-linguagem
Laboratórios suportados:
- `javascript`: Implementações baseadas em objetos e arrays dinâmicos.
- `python`: Implementações focadas em classes e referências.

## 🔄 Requisitos de Laboratório
Como as estruturas de dados (ex: Listas Ligadas) dependem de classes:
- O `template` deve fornecer o esqueleto da classe.
- O `test` deve verificar se os atributos internos (ex: `this.next` ou `self.next`) foram inicializados corretamente conforme a instrução.
