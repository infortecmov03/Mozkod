# Área de Conhecimento: Física para Computação

Este diretório contém os dados de conteúdo para a área de Física no Nível 2.

## Foco

Esta área explora os princípios da física que são aplicados diretamente no desenvolvimento de software, especialmente em simulações, desenvolvimento de jogos e computação gráfica.
