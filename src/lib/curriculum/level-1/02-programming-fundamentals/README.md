# 🐍 Programming Fundamentals (Multi-language)

Este módulo é o coração da lógica de programação e suporta múltiplas linguagens.

## 🌐 Suporte de Linguagens
Este módulo aceita laboratórios em:
- `javascript`
- `python`
- `java`
- `cpp`

## 📁 Organização de Laboratórios
Os exercícios práticos estão organizados por linguagem dentro da pasta `laboratory/`.
- Para adicionar uma nova linguagem, cria a subpasta correspondente (ex: `laboratory/rust/`).
- Regista os novos exercícios no objeto `practice` dentro de `src/lib/curriculum/level-1/02-programming-fundamentals/laboratory.ts`.

## ✅ Validação Multi-linguagem
Certifica-te que o campo `test` do objetivo é compatível com a sintaxe da linguagem específica do laboratório.
