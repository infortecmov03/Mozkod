# 💻 CS Core (Ciência da Computação)

Este módulo foca na base teórica e conceptual da computação.

## 📁 Estrutura
- `theory/`: Lições sobre binário, lógica booleana e arquitetura.
- `quizzes/`: Validações de conhecimento técnico.
- `laboratory/`: Exercícios práticos usando a linguagem `concept`.

## 🧪 Laboratórios de Conceito
Diferente dos laboratórios de código, aqui validamos a **lógica pura**. 
- Utilize o tipo `language: "concept"` no objeto do exercício.
- O campo `test` deve validar strings ou números que representem o resultado da análise lógica do aluno.
