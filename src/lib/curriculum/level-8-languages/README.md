## 🔄 Regra de Ouro: Herança de Projeto
Neste nível, os laboratórios usam `isProjectPart: true`.
- O código salvo pelo aluno na aula 1 é carregado automaticamente na aula 2.
- Isso permite a construção de um **Projeto Master** cumulativo.
