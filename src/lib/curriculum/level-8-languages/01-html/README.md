# 📄 HTML Master - Guia de Expansão

Trilha focada em arquitetura de documentos e acessibilidade.

## 🛠️ Laboratórios do Projeto Master
Os laboratórios devem evoluir o ficheiro `index.html` do aluno:
1. `html-p1`: Estrutura base.
2. `html-p2`: Semântica.
3. `html-p3`: Metadados.

## ✅ Validação
Use o campo `test` para verificar se o aluno adicionou as tags ou atributos solicitados (ex: `test: "role='main'"`).
