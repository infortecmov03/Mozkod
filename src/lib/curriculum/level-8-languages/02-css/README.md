# 🎨 CSS Master - Guia de Expansão

Trilha focada em Design Systems e performance visual.

## 🔄 Herança
Este módulo herda o HTML construído pelo aluno no módulo de HTML Master se as lições estiverem marcadas com `isProjectPart: true`.

## 🧪 Laboratórios
Os exercícios devem fornecer `htmlTemplate` se precisarem de uma estrutura específica para o teste de estilo.
